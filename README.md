# 🎮 Number Guessing Game in C

## 📌 About
A fun console-based number guessing game built using C.

## 🚀 Features
- Random number generation
- Difficulty levels
- Limited attempts
- Score tracking

## 🛠️ Tech Used
- C Programming
- Standard Library

## ▶️ How to Run
gcc main.c -o game
./game
